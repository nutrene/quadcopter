# 課題研究 - 無人航空機の制作

<h2>コントローラー設計</h2>
<strong>左アナログスティック上</strong> -> 前進<br>
左アナログスティック下 -> 後退<br>
左アナログスティック右 -> 右移動<br>
左アナログスティック左 -> 左移動<br>
右アナログスティック上 -> 上昇<br>
右アナログスティック下 -> 下降<br>
右アナログスティック右 -> 右回転<br>
右アナログスティック左 -> 左回転<br>
